Testando Media Queries

Media queries são uma funcionalidade do CSS que permitem adaptar o layout de uma página web para diferentes tamanhos de tela e dispositivos. Elas são amplamente utilizadas no design responsivo, garantindo que o conteúdo seja exibido de forma adequada em diversos contextos, como em celulares, tablets, laptops ou desktops.

A ideia básica por trás das media queries é detectar as características do dispositivo, como a largura e altura da tela, a resolução, a orientação (retrato ou paisagem) e até mesmo as capacidades de cores, e aplicar estilos CSS específicos com base nessas características. Isso significa que você pode criar um único site que se adapta automaticamente ao dispositivo do usuário, em vez de precisar desenvolver várias versões separadas do mesmo site.
