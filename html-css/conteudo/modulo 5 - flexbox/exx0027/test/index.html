Flexbox Actives

Flexbox é um modelo de layout introduzido no CSS3 que facilita a organização de elementos em uma página web, tornando mais simples o alinhamento e distribuição de itens em contêineres. Ele foi projetado para resolver os problemas de alinhamento e distribuição de espaço em layouts responsivos, ou seja, aqueles que se ajustam a diferentes tamanhos de tela de maneira fluida.
Quando você usa o display: flex em um contêiner, todos os elementos filhos dessecontêiner se tornam flex items, e o comportamento deles é regido pelaspropriedades do Flexbox. A ideia central do Flexbox é organizar os itens ao longode um eixo principal e um eixo cruzado. Por padrão, o eixo principal vai daesquerda para a direita (horizontal), e o eixo cruzado vai de cima para baixo(vertical), mas isso pode ser ajustado conforme necessário.
